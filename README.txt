The Orthonormality Transformation based Measure of Relative importance 

1. Z: minimal transformation + A: identity mapping -> Johnson, R. M. (1966). The minimal transformation to orthonormality. Psychometrika, 31(1), 61-66.

2. Z: minimal transformation + A: regression coefficients based reallocation -> (GCD) Green, P. E., Carroll, J. D., & DeSarbo, W. S. (1978). A new measure of predictor variable importance in multiple regression. Journal of Marketing Research, 15(3), 356-360.

3. Z: minimal transformation + A: correlation coefficients based reallocation -> (Relative Weight) Johnson, J. W. (2000). A heuristic method for estimating the relative weight of predictor variables in multiple regression. Multivariate behavioral research, 35(1), 1-19.
